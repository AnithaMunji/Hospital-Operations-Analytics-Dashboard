# Hospital Operations Analytics Dashboard | Python, MySQL, Power BI

## Project Overview
The Hospital Operations Analytics Dashboard is a healthcare analytics project developed using Python, MySQL, and Power BI to analyze patient outcomes, treatment costs, readmission rates, billing performance, and operational efficiency across hospital departments.

The project helps hospitals gain actionable insights by transforming raw healthcare data into meaningful visualizations and KPI-driven reports for operational and financial decision-making.

---

# Objectives
- Analyze patient readmission trends
- Monitor treatment costs and billing performance
- Track operational efficiency across departments
- Identify hospital performance trends
- Build interactive dashboards for data-driven decision-making
